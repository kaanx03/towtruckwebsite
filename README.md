# Borcelle Çekici Servisi

A responsive, multilingual tow truck service website for Istanbul, Turkey. Built with vanilla HTML, CSS, and JavaScript — no frameworks or build tools required.

**Live site:** https://cekicistanbul.netlify.app/

## Features

- **Multilingual** — Turkish, English, and Russian via a flag-based language switcher
- **Typewriter hero animation** with CTA buttons (call & WhatsApp location share)
- **Service cards** — accident tow, breakdown, roadside assistance, garage transfer, motorcycle, and heavy vehicle
- **Photo gallery** with modal lightbox
- **Expandable district list** — shows all Istanbul districts served
- **Sticky navbar** with mobile hamburger menu
- **Floating call button** and back-to-top button
- Fully responsive layout (mobile-first)

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [Font Awesome 6](https://fontawesome.com/) for icons
- [Google Fonts — Inter](https://fonts.google.com/specimen/Inter)
- Hosted on [Netlify](https://netlify.com/)

## Project Structure

```
towtruckwebsite/
├── index.html       # Main page
├── styles.css       # All styles
├── script.js        # Interactivity & translations
├── content.json     # Translation strings
└── images/          # Logo, tow truck photos, Istanbul photo
```

## Getting Started

No build step needed — just open `index.html` in a browser or serve with any static file server:

```bash
npx serve .
```
